- Uses pinned actions: checkout@v4, configure-pages@v5, jekyll-build-pages@v1, upload-pages-artifact@v3, deploy-pages@v4

- Deploys to `github-pages` environment with URL output

### Key decisions
- `cancel-in-progress: false` — consistent with release workflow philosophy: never cancel a deployment mid-flight
- Source set to `./docs` so only the docs directory is built, not the entire repo
- Added `workflow_dispatch` for manual re-deploys without needing a docs commit
