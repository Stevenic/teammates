## Task: Robust handoff detection

Fixed handoff detection in `parseMessageProtocol` ‚Äî was only scanning first 5 lines for `TO:`, but agent output (Claude Code) buries the response deep in tool-call output. Now uses 3-pass detection:

1. **[handoff:] marker** ‚Äî `[handoff:<teammate>]` anywhere in output (most explicit)
2. **Quick scan** ‚Äî `TO:` in first 5 lines (clean output, existing behavior)
3. **Deep scan** ‚Äî `TO: <known_teammate>` anywhere in output (catches buried responses)

Also updated the output protocol prompt to instruct agents to include the `[handoff:]` marker when handing off.
