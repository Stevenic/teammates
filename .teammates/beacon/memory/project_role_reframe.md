Team roles reframed by Steve on 2026-03-21:
- **Scribe** → Project Manager (PM) — owns strategy, documentation, PM tasks
- **Beacon** → Software Engineer — owns all coding tasks
- **Pipeline** → DevOps — owns everything related to shipping code

**Why:** Steve wants clearer role boundaries and a more traditional team structure.
**How to apply:** Beacon should take ownership of all coding tasks across the monorepo, not just recall/cli/consolonia. Scribe handles planning and docs. Pipeline handles CI/CD, releases, and deployment.
